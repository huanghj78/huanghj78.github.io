数据结构 - Double Hashing

Created2022-08-06|数据结构

Consider a hash table with M = 64 bins. Given a 12-bit unsigned number, take the lowest 6 bits as the primary hash function to get the bin number, and the highest 6 bits as the secondary hash function to get the jump size.

  • Determine the array structures for a hash table
  • Hash a list of 40 random 12-bit positive numbers
  • Calculate the average number of probes

Description of Work 08-1 and data structures applied

此实验需要对一组给定的十二位无符号数进行双哈希存到哈希表中，其中第一次哈希将低六位作为初始下标，第二次哈希将高六位作为跳步的步长。
运用的数据结构为数组，及一个结构体，结构体中包含的即是此数组，加上装载的个数和探测次数。
