数据结构 - Huffman Coding

Created2022-08-06|数据结构
  • Determine the data structures for a binary Huffman tree
  • Implement the algorithm of creating a binary Huffman tree for a given postive integer sequence
  • Design and implement a Huffman coding prototype

Description of Lab Work 07-1 and data structures applied

本实验需要做的是根据所给的一个序列（代表权值），生成对应的哈夫曼树，并根据生成的哈夫曼树生成各个权值所对应的哈夫曼编码。
需要用到的数据结构为数组，用静态数组模拟二叉树，为方便计算，第一个元素留空，即使用的下标从1开始。数组的数据类型为自定义的结构体Hnode，其中包含双亲节点及左右儿子的下标和该元素的权值。而生成的哈夫曼树的编码也是由一个静态线性结构存储，其数据类型为自定义的结构体Hcode，其中包含一个存放该元素对应的编码的数组，及编码位数。
生成哈夫曼树首先需要获取数据个数n，然后循环n次，根据哈夫曼树的生成规则，每次找出最小的两个节点，合并生成新节点作为该两个节点的左右儿子，记录下对应的下标即可。
而生成哈夫曼编码则要利用生成的哈夫曼树，对数组的前n个元素逐个进行编码，具体的编码规则为判断该元素为左儿子还是右儿子，依照左0右1的规则进行编码，然后访问该元素的双亲节点，重复此步骤直到根节点。
