    (1) 输入条件的自然语义陈述；
    (2) 输出结果的自然语义陈述；
    (3) 用命题逻辑形式描述实现上述输入-输出过程所应用的判定规则，并写出获得输出结果的推理演算过程。

第6列：

(1) 输入条件的自然语义陈述：

​ 输入为11010，根据原因清单可知表示C1售货机可找零，C2投入1元硬币，C4按下橙子按钮

(2) 输出结果的自然语义陈述：

​ 输出为00110，根据结果清单可知表示E23退还5角硬币，E24送出橙子饮料

(3) 用命题逻辑形式描述实现上述输入-输出过程所应用的判定规则，并写出获得输出结果的推理演算过程:

​ 根据因果图，要得到E23和E24，应该应用的判定规则为：

  • T13 => E23

  • C1 ∧ T11 => T13

  • C2 ∧ T12 => T11

  • C4 ∨ C5 => T12

  • T14 => E24

  • C3 ∨ T13 => T14

推理演算过程：由C1 = C2 = C4 = 1有：

对于E23

  • T12 = C4 ∨ C5 = 1

  • T11 = C2 ∧ T12 = 1

  • T13 = C1 ∧ T11 = 1

  • E23 = T13 = 1

对于E24

  • T14 = C3 ∨ T13 = 1

  • E24 = T14 = 1

故由此可得E23 = E24 = 1

第23列

(1) 输入条件的自然语义陈述：

​ 输入为01001，根据原因清单可知表示~C1售货机不可找零，C2投入1元硬币，C5按下啤酒按钮

(2) 输出结果的自然语义陈述：

​ 输出为11000，根据结果清单可知表示E21零钱找完，E22退还1元硬币

(3) 用命题逻辑形式描述实现上述输入-输出过程所应用的判定规则，并写出获得输出结果的推理演算过程:

​ 根据因果图，要得到E21和E22，应该应用的判定规则为：

  • ~C1 <=> E21

  • ~C1 ∧ T11 => E22

  • C2 ∧ T12 => T11

  • C4 ∨ C5 => T12

推理演算过程：由C1 = 0，C2 = C5 = 1有：

对于E21

  • E21 = ~C1 = 1

对于E22

  • T12 = C4 ∨ C5 = 1

  • T11 = C2 ∧ T12 = 1

  • E22 = ~C1 ∧ T11 = 1

故由此可得E21 = E22 = 1
