软件测试-Assignment5 软件复杂度

Created2022-07-30|软件测试

  1. 计算下列代码片段的 Halstead 复杂度的11项内容：

声明：括号、分号不计入操作符

  • 首先，设 n1 表示程序中不同的操作符个数，n2 表示程序中不同的操作数个数，N1 表示程序中出现的操作符总数，N2 表示程序中出现的操作数总数，则上面的代码片段中：

    • n1 = 11，分别为： if、<、=、+、-、return、dayray、int、*、/、%
    • n2 = 13，分别为：month、year、day、3、12、1、26、10、4、6、100、400、7
    • N1 = 22
    • N2 = 22

    1. Halstead程序词汇表长度Program vocabulary：n = n1 + n2 = 24

    2. Halstead 程序长度或简单长度 Program length：N = N1 + N2 = 44

    3. 程序的预测长度 Calculated program length：N^ = n1log2n1 + n2 log2n2 = 11* log2(11) + 13*log2(13) = 86.15946414

    4. 程序体积或容量 Volume: V = Nlog2(n) = 44*log2(24) = 201.73835

    5. 程序级别 Level: L^ = (2/n1) * (n2/N2) = (2 / 11) * (13 / 22) = 13 / 121 = 0.1074380165

    6. 程序难度 Difficulty: D = 1/L^ = 121 / 13 = 9.307692308

    7. 编程工作量 Effort: E = V * D = V/L^ = 201.73835 * 9.307692308 = 1877.718488

    8. 语言级别: Lʹ = L^ * L^ * V = (13 / 121) * (13 / 121) * 201.73835 = 2.328651127

    9. 编程时间 (hours): T^ = E / (S * f)，这里 S = 60 * 60, f = 18，故T^ = 1877.718488 / (60 * 60 * 18) = 0.02897713716

    10. 在上面代码片段中，共有4条语句，故平均语句大小: N / 语句数 = 44 / 4 = 11

    11. 程序中的错误数预测值: B = V / 3000 = 201.73835 / 3000 = 0.06724611667
